# space-crew
A multiplayer videogame, maybe.

## How to run your own server

Install [Node.js](https://nodejs.org/), clone the repository and run:

```bash
npm install
npm run build
```

Then start a server with:

```bash
node server
```

And open `localhost:3000` in your browser.
